# Report

## Introduction

The report is a markdown file generated by the test executive upon completion of a test sequence. This functionality can be enabled and disabled through the settings of the test executive. The report can be added to using the "Report" step type detailed in the [steps.md](steps.md) file.

## Report Structure

```md
# Report Title

## Information

Sequence Version: {version}
Operator: {operator}

## Tests

| Test | Result |
| ---- | ------ |
| {test} | {result} |
| {test} | {result} |
| {test} | {result} |

## Results

{timestamp}: {sequence added report}
{timestamp}: {sequence added report}
{timestamp}: {sequence added report}
{timestamp}: {sequence added report}
```
