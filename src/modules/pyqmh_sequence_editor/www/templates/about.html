pyqmh Sequence Editor

The sequence editor provides a visual environment for creating and maintaining pyqmh sequence JSON files. Steps are represented as connected flow nodes, with focused editing for parameters and variables.

File operations are handled by the module over protocol request messages so the UI remains decoupled from direct cross-module calls and follows the queue-based pyqmh architecture.
